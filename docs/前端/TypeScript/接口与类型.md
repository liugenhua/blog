# TypeScript 接口与类型

## Interface

```ts
interface User {
  id: number
  name: string
  email?: string   // 可选属性
  readonly createdAt: Date  // 只读
}

const user: User = {
  id: 1,
  name: "张三",
  createdAt: new Date(),
}
```

## Type

```ts
type Point = {
  x: number
  y: number
}

type ID = string | number  // 联合类型
type Status = "active" | "inactive"  // 字面量类型
```

## Interface vs Type

| 特性 | Interface | Type |
|------|-----------|------|
| 扩展 | `extends` | `&` 交叉类型 |
| 合并声明 | ✅ 支持 | ❌ 不支持 |
| 原始类型别名 | ❌ | ✅ |
| 联合类型 | ❌ | ✅ |

> 原则：能用 interface 就用 interface，需要联合/交叉类型时用 type。
