## <font style="color:#000000;">2. KNN 实现分类任务</font>
+ <font style="color:#000000;">学会使用</font><font style="color:#000000;"> </font><font style="color:#000000;">KNeighborsClassifier</font><font style="color:#000000;"> </font><font style="color:#000000;">做分类</font>
+ <font style="color:#000000;">知道分类任务的目标是预测“类别”</font>
+ <font style="color:#000000;">理解 KNN 的核心思想：看离目标最近的 K 个邻居，大多数属于哪一类，就预测成哪一类</font>

<font style="color:#000000;">代码流程：</font>

1. <font style="color:#000000;">导入模型</font>
2. <font style="color:#000000;">准备训练数据</font><font style="color:#000000;"> </font><font style="color:#000000;">x_train</font>
3. <font style="color:#000000;">准备标签</font><font style="color:#000000;"> </font><font style="color:#000000;">y_train</font>
4. <font style="color:#000000;">创建 KNN 分类器</font>
5. <font style="color:#000000;">调用</font><font style="color:#000000;"> </font><font style="color:#000000;">fit()</font><font style="color:#000000;"> </font><font style="color:#000000;">训练</font>
6. <font style="color:#000000;">调用 </font><font style="color:#000000;">predict()</font><font style="color:#000000;"> 预测</font>

<font style="color:#000000;">核心代码</font>

```python
from sklearn.neighbors import KNeighborsClassifier

x_train = [[0], [1], [2], [3]]
x_test = [[4]]
y_train = [0, 0, 0, 1]

knn_model = KNeighborsClassifier(n_neighbors=2)
knn_model.fit(x_train, y_train)
y_pred = knn_model.predict(x_test)
print(y_pred)
from sklearn.neighbors import KNeighborsClassifier

x_train = [[0], [1], [2], [3]]
x_test = [[4]]
y_train = [0, 0, 0, 1]

knn_model = KNeighborsClassifier(n_neighbors=2)
knn_model.fit(x_train, y_train)
y_pred = knn_model.predict(x_test)
print(y_pred)

```

<font style="color:#000000;">重点理解：</font>

+ <font style="color:#000000;">x_train</font><font style="color:#000000;"> </font><font style="color:#000000;">是特征数据</font>
+ <font style="color:#000000;">y_train</font><font style="color:#000000;"> </font><font style="color:#000000;">是标签数据</font>
+ <font style="color:#000000;">n_neighbors=2</font><font style="color:#000000;"> </font><font style="color:#000000;">表示找最近的 2 个邻居</font>
+ <font style="color:#000000;">这个例子本质是在做“二分类”</font>

<font style="color:#000000;">补充理解：</font>

+ <font style="color:#000000;">如果邻居中类别票数相同，结果可能和底层规则有关，所以实际使用中常设置奇数个邻居，比如</font><font style="color:#000000;"> </font><font style="color:#000000;">3</font><font style="color:#000000;">、</font><font style="color:#000000;">5</font>

## <font style="color:#000000;">  
</font><font style="color:#000000;"> 3. KNN 实现回归任务</font>
<font style="color:#000000;">文件：02_knn实现回归任务.py</font>

<font style="color:#000000;">学习内容：</font>

+ <font style="color:#000000;">学会使用 </font><font style="color:#000000;">KNeighborsRegressor</font><font style="color:#000000;"> 做回归</font>
+ <font style="color:#000000;">知道回归任务的目标是预测“具体数值”</font>
+ <font style="color:#000000;">理解 KNN 回归思想：找到最近的 K 个样本，对它们的目标值做平均或加权平均</font>

```python
from sklearn.neighbors import KNeighborsRegressor

x_train = [[0], [1], [2], [3]]
x_test = [[4]]
y_train = [70, 80, 100, 110]

knn_model = KNeighborsRegressor(n_neighbors=3)
knn_model.fit(x_train, y_train)
y_pred = knn_model.predict(x_test)
print(y_pred)

```

<font style="color:#000000;">重点理解：</font>

+ <font style="color:#000000;">分类预测的是类别，比如“是不是垃圾邮件”</font>
+ <font style="color:#000000;">回归预测的是连续值，比如“房价是多少”</font>
+ <font style="color:#000000;">n_neighbors=3</font><font style="color:#000000;"> </font><font style="color:#000000;">表示参考最近的 3 个邻居</font>
+ <font style="color:#000000;">预测结果 </font><font style="color:#000000;">96.666...</font><font style="color:#000000;"> 说明 KNN 回归会根据邻居的目标值综合得到一个数值结果</font>

## <font style="color:#000000;">4. 特征数据归一化</font>
**<font style="color:#000000;">学习内容：</font>**

+ **<font style="color:#000000;">学会使用 </font>****<font style="color:#000000;">MinMaxScaler</font>**
+ **<font style="color:#000000;">理解归一化的作用：把不同范围的数据缩放到统一范围，通常是 </font>****<font style="color:#000000;">[0, 1]</font>**
+ **<font style="color:#000000;">明白归一化常用于特征数值差异比较大的情况</font>**

```python
from sklearn.preprocessing import MinMaxScaler

x_train = [
    [90, 2, 10, 40],
    [60, 4, 15, 45],
    [75, 3, 13, 46]
]

model = MinMaxScaler()
new_x_train = model.fit_transform(x_train)
print(new_x_train)

```

<font style="color:#000000;">重点理解：</font>

+ <font style="color:#000000;">fit_transform()</font><font style="color:#000000;"> </font><font style="color:#000000;">= 先学习数据规律，再完成转换</font>
+ <font style="color:#000000;">归一化公式：</font>

`<font style="color:#000000;">X' = (X - Xmin) / (Xmax - Xmin)</font>`

+ <font style="color:#000000;">归一化后，不同特征的量纲差异会减小，更适合距离类算法，比如 KNN</font>

## <font style="color:#000000;">5. 特征数据标准化</font>
<font style="color:#000000;">学习内容：</font>

+ <font style="color:#000000;">学会使用 </font><font style="color:#000000;">StandardScaler</font>
+ <font style="color:#000000;">理解标准化的作用：把数据处理成均值为 0、标准差为 1</font>
+ <font style="color:#000000;">知道标准化在很多机器学习任务中更常用，所以文件里也标了“推荐”</font>

```python
from sklearn.preprocessing import StandardScaler

x_train = [
    [90, 2, 10, 40],
    [60, 4, 15, 45],
    [75, 3, 13, 46]
]

model = StandardScaler()
new_x_train = model.fit_transform(x_train)
print(new_x_train)

```

<font style="color:#000000;">  
</font><font style="color:#000000;"> </font><font style="color:#000000;">重点理解：</font>

+ <font style="color:#000000;">标准化公式：</font>

`<font style="color:#000000;">X' = (X - mean) / std</font><font style="color:#000000;background-color:rgb(36, 37, 38);">  
</font>`

+ <font style="color:#000000;">标准化后的数据不一定在</font><font style="color:#000000;"> </font><font style="color:#000000;">[0, 1]</font>
+ <font style="color:#000000;">适合数据分布较复杂的场景，对很多模型效果更稳定</font>
